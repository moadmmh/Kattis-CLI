# Kattis-CLI
Kattis-CLI is a command-line interface tool for [Kattis OJ](https://open.kattis.com/) .

### Installation
`sudo npm install kattis-cli -g`

### Usage
  - #### Configurtion 
    `kattis config`
  
    Then you need to fill your 
  
    `Email :`
  
    `Password :`
  
    `Username :`
  
  - #### Submission 
  
    `kattis submit -p <pblmid> -f <sourcefile>`
